# Rust-Programming-in-Easy-Steps
Rust Programming in Easy Steps, Richard Urwin © 2026 by In Easy Steps Limited.

My projects here are examples I followed from the Rust in Easy Steps book.

I will often add a few things here and there to the examples for ease of use, etc., so the source code may not be the same as seen in the book. For example, in the pontoon card game, I added a loop for continuous play.

7/6/2026 - Sharing my thoughts as I am halfway through the book (ch.6 completed of 12). Rust seems to be quite a bit more difficult than python or java, however, I'm not sure if it's the way the book is written/orginized or it's the language itself. In some chapters, the author has you do things wrong on purpose, as if to say something like "you can't do this" or "this won't work", which I find odd, but perhaps it's just simply to show you the error messages in the terminal. I'll share my thoughts again when I complete the book.
