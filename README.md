# Rust-Programming-in-Easy-Steps
Rust Programming in Easy Steps, Richard Urwin © 2026 by In Easy Steps Limited.

My projects here are examples I followed from the Rust in Easy Steps book.

I will often add a few things here and there to the examples for ease of use, etc., so the source code may not be the same as seen in the book. For example, in the pontoon card game, I added a loop for continuous play.

7/6/2026 - Sharing my thoughts as I am halfway through the book (ch.6 completed of 12). Rust seems to be quite a bit more difficult than python or java, however, I'm not sure if it's the way the book is written/organized or it's the language itself. In some chapters, the author has you do things wrong on purpose, as if to say something like "you can't do this" or "this won't work", which I find odd, but perhaps it's just simply to show you the error messages in the terminal. I'll share my thoughts again when I complete the book.

7/22/2026 - Having finished the book today, I feel like much more practice is needed to try to start a new project on my own. However, I have achieved my goal of understanding how Rust works, what it's used for, and experiencing the syntax. Rust is a complicated programming language, not very intuitive, and the syntax is not similar to anything I have worked with before. I do like that there is not really a separate error handling angle, you include it with everything you do. As far as the book Rust Progamming in Easy Steps goes, the author has made a few mistakes throughout the book (other than the occasional non-impactful typos), like forgetting to include an important simple line of code that, which I had to reference the author's examples downloaded from the free resources on ineasysteps.com to get my project working.

Overall, I enjoyed learning something new from this Rust book and it's short easy steps approach, along with the other 'in easy steps' books in my library. They're short and to the point, giving you a quick hands-on sampling to help you better decide if you would like to continue learning the subject.
